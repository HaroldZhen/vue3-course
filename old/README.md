### 依據使用者需求完成此範例：

製作 loading 效果

前台頁面串接 API

頁面模板
產品列表
單一產品細節
加入購物車
購物車列表
刪除購物車項目（單一、全部）
調整購物車產品數量
結帳付款
注意：

新增相同產品到購物車時需累加項目
送出訂單後，購物車需要清除原本項目
購物車無產品時不建議發出結帳請求
前台頁面表單驗證（必要完成），驗證內容包含：

姓名：必填
Email：必填 / 需要符合格式 / input type 為 email
電話：必填 / 超過 8 碼 / input type 為 tel
地址：必填
留言：非必填
